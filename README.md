# 🚀 Cosmic Defender

**Cosmic Defender** é um jogo estilo **Space Shooter** desenvolvido com tecnologias modernas da web.

O jogador controla uma nave espacial e precisa destruir inimigos enquanto desvia de ataques para alcançar a maior pontuação possível.

O projeto foi construído do zero utilizando **Canvas API** e **React Hooks**, sem o uso de engines externas de jogos.

---

# 🎮 Funcionalidades do Jogo

No jogo você pode:

- 🚀 Controlar uma nave espacial
- 🔫 Atirar contra inimigos
- 🛸 Desviar de ataques
- ⭐ Acumular pontuação
- 🏆 Salvar automaticamente o **High Score**

# 🌐 APIs do Navegador

O jogo utiliza recursos nativos do navegador:

- **Canvas API** → Renderização gráfica 2D do jogo  
- **requestAnimationFrame** → Loop do jogo a 60 FPS  
- **Keyboard Events** → Controles por teclado  
- **Touch Events** → Controles em dispositivos móveis  
- **localStorage** → Salvamento do recorde do jogador  

---

# 🧠 Conceitos de Programação Aplicados

Este projeto utiliza diversos padrões usados no desenvolvimento de jogos:

- **Custom Hooks** → `useGame()` encapsula toda a lógica do jogo  
- **Game Loop Pattern** → Separação entre lógica e renderização  
- **Collision Detection (AABB)** → Detecção de colisões entre objetos  
- **Object Pooling** → Gerenciamento eficiente de entidades  
- **State Machine** → Estados do jogo (menu, jogando, pausado, game over)  
- **Component Composition** → Separação em componentes reutilizáveis
